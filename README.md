# 🎱 Bingo Maciej PWA

Nowoczesna odsłona klasycznej gry Bingo, stworzona w technologii **React** z wykorzystaniem **Vite** i **Tailwind CSS**. Aplikacja jest pełnoprawnym **PWA**, co oznacza, że możesz ją zainstalować na swoim telefonie lub komputerze i grać nawet w trybie offline!

![Bingo Maciej Icon](public/icon-512x512.png)

## 🚀 Cechy gry
- **Modern UI**: Elegancki interfejs w stylu 3D Premium.
- **PWA Ready**: Możliwość instalacji jako aplikacja mobilna.
- **Mechanika Real-Time**: Możesz zaznaczyć pole tylko dla ostatnio wylosowanej kuli - wymaga refleksu!
- **Konfetti**: Efektowne świętowanie wygranej.
- **Responsive**: Gra idealnie dopasowuje się do każdego ekranu.

## 🛠️ Technologie
- **React 19**
- **Vite 6**
- **Tailwind CSS**
- **Vite PWA Plugin**

## 📦 Instalacja i uruchomienie lokalne

1. Sklonuj repozytorium:
   ```bash
   git clone https://github.com/TWOJA-NAZWA/gra-bingo.git
   ```
2. Zainstaluj zależności:
   ```bash
   npm install
   ```
3. Uruchom wersję deweloperską:
   ```bash
   npm run dev
   ```
4. Zbuduj wersję produkcyjną:
   ```bash
   npm run build
   ```

## 🌐 Publikacja na GitHub Pages
Projekt jest skonfigurowany pod **GitHub Actions**. Po wypchnięciu kodu na gałąź `main`, Twoja gra zostanie automatycznie opublikowana pod adresem:
`https://TWOJA-NAZWA.github.io/gra-bingo/`
